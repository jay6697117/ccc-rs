# Phase 12: TUI & Agent Integration — 深度集成与流式渲染

## 目标
实现 TUI 与后台 Agent 的无缝衔接，提供毫秒级的流式响应反馈。

## 核心功能
1. **流式数据绑定**：
   - 建立线程安全的异步管道，将 Agent 的流式 ContentBlock 实时推送至 UI 渲染缓冲区。
2. **异步任务编排**：
   - 在 `tokio::spawn` 中管理 AI 推理任务，避免长时间阻塞 UI 主线程。
3. **自动滚动与长对话处理**：
   - 实现视口裁剪与平滑滚动算法，处理数万行的上下文对话历史。
4. **MCP 动态注册**：
   - 在 TUI 启动时自动扫描并连接配置中的 MCP 服务器，并将其工具注入 Agent 注册表。
