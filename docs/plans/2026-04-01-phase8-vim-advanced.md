# Phase 8: ccc-vim — 高级 Vim 状态机实现

## 目标
在 Rust 中重现完整的 Vim 交互逻辑，支持复杂的组合操作。

## 核心功能
1. **Motions (运动)**：
   - 基础：`h/j/k/l`, `w/b/e`。
   - 高级：`f/t/F/T`, `0/$`, `gg/G`, `%/` (括号匹配)。
2. **Operators (操作符)**：
   - `d` (删除), `c` (修改), `y` (复制), `v` (可视化模式)。
3. **Text Objects (文本对象)**：
   - `iw/aw` (单词), `ip/ap` (段落), `i(/a(` (括号内容)。
4. **寄存器与宏**：
   - 支持无名寄存器与命名寄存器存储剪贴板内容。

## 依赖项
- `ccc-core` (类型定义)
