# Phase 10: ccc-cli — 命令行入口与解析

## 目标
基于 `clap` 构建功能完备的命令行入口，支持子命令与全局参数。

## 核心功能
1. **子命令架构**：
   - `ccc login`：执行 OAuth 流程。
   - `ccc chat`：启动 TUI 交互模式。
   - `ccc config`：查看或修改本地配置。
2. **参数解析**：
   - 支持 `--model`, `--max-tokens` 等动态覆盖。
   - 环境变量自动映射（如 `ANTHROPIC_API_KEY`）。
3. **管道支持**：
   - 允许通过管道输入内容：`cat script.py | ccc chat "Explain this"`。

## 依赖项
- `clap` (derive features)
- `ccc-auth`, `ccc-tui`
