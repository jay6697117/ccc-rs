# Phase 7: Implement ccc-tui — Ratatui based interface

## Objective
Implement the Terminal User Interface using `ratatui` and `crossterm`, providing a rich, interactive experience equivalent to the original `ink` based interface.

## Tasks
1. [ ] Set up `ccc-tui` crate and dependencies.
2. [ ] Implement terminal event loop (keyboard/mouse handling).
3. [ ] Design and implement the UI layout (header, conversation area, input area, task list).
4. [ ] Implement Vim-style keybindings using `ccc-vim`.
5. [ ] Connect `ccc-tui` to `ccc-agent` for real-time interaction.
6. [ ] Implement streaming output rendering.
7. [ ] Add support for syntax highlighting and markdown rendering.
