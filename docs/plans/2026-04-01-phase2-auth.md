# Phase 2: ccc-auth — 身份验证与令牌管理

## 目标
实现安全且自动化的身份验证系统，支持 OAuth PKCE 和 API Key 方案。

## 核心功能
1. **OAuth 2.0 PKCE 流程**：
   - 生成 Code Verifier 和 Challenge。
   - 浏览器重定向授权。
   - 令牌交换与存储。
2. **安全存储**：
   - 集成操作系统 Keyring (macOS/Linux/Windows)。
   - 加密存储 Access Token 和 Refresh Token。
3. **令牌刷新机制**：
   - 自动检测过期并执行刷新请求。
4. **API-key 解析**：
   - 兼容环境变量、配置文件及动态解析。

## 依赖项
- `ccc-core` (类型与错误处理)
- `keyring-rs` (平台密钥管理)
- `reqwest` (验证请求)
