# Phase 5: ccc-tools — 基础工具实现

## 目标
实现 AI 代理可以调用的核心本地工具集，确保安全与可观测性。

## 核心功能
1. **文件系统操作**：
   - `Read`：分块读取文件内容。
   - `Edit`：基于精确字符串替换或 Patch 的编辑。
   - `Write`：原子性地写入新文件。
2. **环境探测**：
   - `Bash`：受限的 Shell 命令执行环境。
   - `Glob/Grep`：快速文件搜索。
3. **任务管理工具**：
   - `TaskCreate/Update`：允许 AI 追踪自身任务进度。
4. **安全性限制**：
   - 路径沙盒化（防止跨目录读写）。
   - 命令白名单机制。

## 依赖项
- `ignore` (尊重 .gitignore 的搜索)
- `tempfile` (安全写入)
